# AI-Driven Customer Churn Prediction

An end-to-end Business Analysis + Machine Learning project 
identifying customer churn drivers in UK financial services, 
with quantified revenue impact and MoSCoW-prioritised 
BA recommendations.

## 🔗 Live Project
👉 https://kinjalshah006.github.io/churn-prediction-project

## 🛠 Built With
- Python · Scikit-learn · Pandas · NumPy
- Random Forest Classifier (86.7% accuracy · AUC-ROC 0.847)
- Power BI-style interactive dashboard
- MoSCoW-prioritised BA recommendations
- Stakeholder analysis & risk matrix
- £ revenue impact ROI model
- GDPR compliance framework

- 20.4% overall churn rate · £4.2M annual revenue at risk
- AI intervention projected to reduce churn by 35%
- Net ROI: £1.245M (692% ROI) · Payback period: ~1.5 months
